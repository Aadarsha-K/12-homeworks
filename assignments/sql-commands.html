SQL (Structured Query Language)

SQL (Structured Query Language) is a standardized language used for querying information from databases. It allows users to create, retrieve, update, and delete data from databases.

Some common SQL Commands

  • CREATE: Creates new database objects like tables, indexes, or views.
  • ALTER: Modifies the structure of existing database objects, such as tables.
  • DROP: Deletes existing database objects like tables or indexes.
  • UPDATE: Modifies existing data in a table based on specified conditions.
  • INSERT: Adds new records (rows) into a table.
  • RENAME: Changes the name of an existing database object, such as a table.
